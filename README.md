brot
====

A Mandelbrot set visualizer.

![step1e-3](https://raw.github.com/sbool/brot/master/step1e-3.png)

Dependencies
------------

- Cython
- NumPy
- Pillow


Running
-------

Before running, compile the Cython module:

    ./setup.py build_ext --inplace

Then run with `./brot.py`. See `./brot.py -h` for help.
